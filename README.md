## Stuff that work

Keypress read
- [x] alphas (a-z, a-z symbols)
- [x] shift + alphas
- [x] ctrl + '@', 'a'-'z', '[', '\', ']', '^', '_'
- [x] alt + alphas
- [x] supr + alphas

Binds
- [x] single key bind
- [x] ctrl + single key bind
- [x] alt + single key bind
- [x] ctrl + alt
- [x] supr + alt + shift + ctrl
